Responsive Web Design

Responsive web design, or RWD is a style of web design that takes into consideration the numerous different devices and screen sizes that people use to view webpages. This approach to web development concentrates on designing pages that can change and adjust their layout depending on to fit screens and optimize user experience.

More on Responsive Web Design

It's important for developers to use RWD in their web pages because of the advances in the world of technology. There are now many different devices capable of web browsing, and with that comes many different screen sizes. It's important to be able to create web sites in which your content is displayed in the way you intended it to be, no matter what device it is being viewed on.

RWD also saves developers time, by allowing you to develop one site capable of adjusting itself instead of multiple different device specific sites.

Tools of RWD

Here are a few examples of elements used to enhance the responsiveness of a web page:

Flexboxes are CSS elements that allows your webpage to easily adapt to changes in screen size while keeping your content displayed the way you want it.

Wireframes are basically sketches used to help plan what the webpage will look like on different screen sizes.
